### 🚀 AI CLI Agents Installer

Interactive Bash/PowerShell installer for terminal-based AI coding agents. Automatically detects and installs Node.js if missing.

* * *

### 🚀 One-Click Installation

### Linux / macOS (Bash)

bash

    git clone https://github.com && cd ai-cli-agents-installer && chmod +x install-ai-agents.sh && ./install-ai-agents.sh
    

Use code with caution.

### Windows (PowerShell)

powershell

    git clone https://github.com; cd ai-cli-agents-installer; .\install-ai-agents.ps1
    

Use code with caution.

* * *

### 🤖 Supported Agents

*   **Claude Code** (`claude`)
*   **Aider** (`aider`)
*   **Gemini CLI** (`gemini`)
*   **Cline** (`cline`)
*   *...and many more available in the interactive menu.*
